# Veridian Health Partners

**Veridian Health Partners** is the company. We build healthcare technology
for jails and correctional facilities.

Our **flagship product** is [**Veridian Motion**](/veridian-motion/product) —
a physical therapy platform for musculoskeletal triage inside custody
settings. Veridian Motion pairs an **inmate kiosk** (structured intake and
posture photos) with a **provider dashboard** where licensed PTs review
AI-assisted triage support and release **PT-reviewed recommendations**. The
product does not provide a diagnosis; clinical authority stays with the
licensed PT.

## Why Veridian Motion

Structured MSK triage and **PT-reviewed bodyweight exercise plans** can improve
inmate wellbeing and help facilities address complaints before they escalate
into costlier outcomes — emergency transfers, chronic disability, repeat sick
calls, and litigation exposure. Veridian Motion is built for that workflow
inside custody, not as a billing or EHR replacement.

<Note>
  The current release does **not** include live video, EHR integration, partner
  API connectivity, or medical billing. Those may be considered in future phases.
</Note>

## Stack at a glance

- **Frontend:** TanStack Start (React 19), installable PWA, USWDS-aligned UI
- **Backend:** Lovable Cloud (Supabase — Postgres, Auth, Storage)
- **Server logic:** TanStack server functions
- **AI:** Multi-provider router with automatic failover; all output is
  non-diagnostic and PT-reviewed before inmate release

## Compliance posture

Veridian Motion is designed with HIPAA-aligned controls, but **is not
HIPAA-certified, CJIS-authorized, or FedRAMP-authorized**. The application
enforces role-based access, audit logging, and data isolation; **hosting
agencies** provide CJIS/FedRAMP/GovCloud environments, MDM kiosk lockdown,
network controls, and personnel screening. See [Security overview](/security/overview).
