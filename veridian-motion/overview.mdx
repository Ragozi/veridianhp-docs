# Platform overview

For product vision and case flow, start with
[Veridian Motion](/veridian-motion/product). This page lists what is **built
today**.

## What it is

| Surface | Users | Purpose |
| --- | --- | --- |
| **Patient portal (kiosk)** | Patients (Patient ID + PIN) | Body map, complaint, questionnaire, up to 4 photos, status and approved plan |
| **Provider dashboard** | Staff (`pt`, `admin`, `it_admin`) | Roster, manual enrollment, assignment queue, review/approve, messaging, PDF |

TanStack Start web app (React 19), installable **PWA**, USWDS-aligned UI,
backed by Supabase (Postgres, Auth, Storage) via Lovable Cloud.

## Built today

**Patient portal**

- Sign-in with **Patient ID + PIN** (staff-provisioned)
- Body-map region selector
- Complaint text
- Plain-language **questionnaire**
- Up to **four guided posture photos**
- Post-submit confirmation; **My plan / progress** status tracking
- Approved exercise plan and shared PT notes **only after PT approval**
- Red-flag inputs → calm **see medical staff** safety action (not diagnosis)

**Provider dashboard**

- Roles: `pt`, `admin`, `it_admin`
- **Manual** patient enrollment (single-facility)
- **Assignment queue** → assessment **review/approve**
- PT notes
- **PDF export**
- **Secure clinician-to-patient messaging** (append-only)

**AI**

- Non-diagnostic triage producing immediate **PT-reviewable exercise plan**
  (name / why / steps / dosage / cautions)
- Licensed PT retains clinical authority

**Security**

- Role-based access control
- Row-level security on every table
- **Hash-chained audit log**
- **Field-level PHI encryption**
- **Brute-force login protection**
- PWA/kiosk hardening
- Staff **TOTP MFA** (available; enforced per deployment)

## What it is not (current release)

- **Not a diagnosis tool**
- **Not live video, EHR, or partner API** — see [Roadmap](/veridian-motion/roadmap)
- **Not SSO** — local staff accounts only today
- **Not bulk roster import or JMS integration** — manual enrollment only
- **Not multi-facility** — single-facility deployments today; multi-facility
  isolation and `warden` oversight **rolling out** — see [Roadmap](/veridian-motion/roadmap)
- **Not HIPAA-certified, CJIS-authorized, or FedRAMP-authorized** — see
  [Security overview](/security/overview)

## PT-review authority

Nothing from AI reaches a patient until a licensed PT **approves** the
assessment. After submit, patients see confirmation and safety guidance only.
